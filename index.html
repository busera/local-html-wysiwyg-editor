How to use

  1. Run the local server with ./start.sh.
  2. Open an HTML file from the vault with Open HTML File.
  3. Edit in the WYSIWYG pane.
  4. Use Save to write back to the same file.

Chrome, Edge, Brave, and Arc support direct save through the File System Access API. Safari and Firefox use manual mode: choose a file, edit it, then download a copy and replace the original file yourself.
